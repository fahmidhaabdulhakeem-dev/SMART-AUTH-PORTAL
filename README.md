# 🔐 Smart Auth Portal
A premium, modern, and highly interactive Login & Registration interface built with pure semantic HTML, vanilla CSS, and vanilla JavaScript. Featuring modern web design paradigms such as **glassmorphism**, dynamic dark/light themes, inline input validations, and a real-time password strength analyzer.

---
## ✨ Features
* **💎 Premium Glassmorphism UI**: Frosted glass effects (`backdrop-filter: blur`), subtle borders, and smooth glowing gradients tailored to create an elegant user experience.
* **🌓 Hardware-Accelerated Theme Toggle**: Seamless dark and light mode toggle. The chosen theme persists across page reloads using browser `localStorage`.
* **📊 Real-time Password Strength Analyzer**: As you type, the password strength is evaluated based on length, casing, numbers, and special symbols, showing live visual progress feedback (Weak ➡️ Medium ➡️ Strong).
* **👁️ Password Visibility Switcher**: Toggle passwords between masked and visible states instantly.
* **⚡ Fluid Navigation Transitions**: Custom page-exit animations when transitioning between the Login and Registration screens.
* **🛡️ Secure Client-Side Validations**: Dynamic inline validations checking email structures, character requirements, and confirm password matching with error alerts.
* **📱 Fully Responsive Design**: Mobile-first design that adapts beautifully to mobile phones, tablets, and desktops.
* **🌀 Interactive Submit States**: Simulated authentication requests featuring loading spinners and visual success toast alerts.
---
## 🛠️ Built With
* **Markup**: HTML5 (Semantic and SEO-friendly structure)
* **Styling**: Vanilla CSS3 (Custom HSL Color Variables, CSS Grid/Flexbox, Keyframe Animations)
* **Logic**: Vanilla ES6 JavaScript (DOM manipulation, Regex validations, localStorage)
* **Icons**: [FontAwesome v6](https://fontawesome.com)
* **Fonts**: [Google Fonts - Outfit](https://fonts.google.com/specimen/Outfit)
